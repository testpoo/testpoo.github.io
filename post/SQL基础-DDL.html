SQL 分类-DDL

SQL 语句主要可以划分为以下 3 个类别。

  • DDL（Data Definition Languages）语句：数据定义语言，这些语句定义了不同的数据段、 数据库、表、列、索引等数据库对象的定义。常用的语句关键字主要包括 create、drop、alter 等。

  • DML（Data Manipulation Language）语句：数据操纵语句，用于添加、删除、更新和查 询数据库记录，并检查数据完整性，常用的语句关键字主要包括 insert、delete、udpate 和 select 等。

  • DCL（Data Control Language）语句：数据控制语句，用于控制不同数据段直接的许可和 访问级别的语句。这些语句定义了数据库、表、字段、用户的访问权限和安全级别。主要的 语句关键字包括 grant、revoke 等。

  • 删除数据库
  -- drop database dbname; # 语法

  • 创建表
  # 创建表
  -- CREATE TABLE tablename (column_name_1 column_type_1 constraints,column_name_2 column_type_2 constraints, ……column_name_n column_type_n constraints) # 语法
  mysql> create table emp(ename varchar(10),hiredate date,sal decimal(10,2),deptno int(2));
  Query OK, 0 rows affected (0.04 sec)

  # 查看表定义
  -- DESC tablename # 语法
  mysql> desc emp;
  +----------+---------------+------+-----+---------+-------+
  | Field    | Type          | Null | Key | Default | Extra |
  +----------+---------------+------+-----+---------+-------+
  | ename    | varchar(10)   | YES  |     | NULL    |       |
  | hiredate | date          | YES  |     | NULL    |       |
  | sal      | decimal(10,2) | YES  |     | NULL    |       |
  | deptno   | int(2)        | YES  |     | NULL    |       |
  +----------+---------------+------+-----+---------+-------+
  4 rows in set (0.02 sec)

  # 查看全面的表定义
  -- SHOW CREATE TABLE TABKENAME # 语法
  mysql> show create table emp;
  +-------+--------------------------------+
  | Table | Create Table                   |
  +-------+--------------------------------+
  | emp   | CREATE TABLE `emp` (
    `ename` varchar(10) DEFAULT NULL,
    `hiredate` date DEFAULT NULL,

  1 row in set (0.00 sec)
  • 删除表
  # 删除表
  -- DROP TABLE tablename # 语法
  mysql> drop table emp;
  Query OK, 0 rows affected (0.02 sec)
  • 修改表
  # 修改表类型
  -- ALTER TABLE tablename MODIFY [COLUMN] column_definition [FIRST | AFTER col_name] # 语法
  mysql>  alter table emp modify ename varchar(20);
  Query OK, 0 rows affected (0.08 sec)
  Records: 0  Duplicates: 0  Warnings: 0

  # 增加表字段
  -- ALTER TABLE tablename ADD [COLUMN] column_definition [FIRST | AFTER col_name] # 语法
  mysql> alter table emp add column age int(3);
  Query OK, 0 rows affected (0.05 sec)
  Records: 0  Duplicates: 0  Warnings: 0

  # 删除表字段
  -- ALTER TABLE tablename DROP [COLUMN] col_name # 语法
  mysql> alter table emp drop column age;
  Query OK, 0 rows affected (0.07 sec)
  Records: 0  Duplicates: 0  Warnings: 0

  # 字段改名
  -- ALTER TABLE tablename CHANGE [COLUMN] old_col_name column_definition [FIRST|AFTER col_name] # 语法
  mysql> alter table emp change sal salary decimal(10,2);
  Query OK, 0 rows affected (0.01 sec)
  Records: 0  Duplicates: 0  Warnings: 0

  # 修改字段排列顺序
  -- 添加字段birth放到ename后面
  mysql> alter table emp add birth date after ename;
  Query OK, 0 rows affected (0.07 sec)
  Records: 0  Duplicates: 0  Warnings: 0
  -- 将salary放到最前面
  mysql> alter table emp modify salary int(3) first;
  Query OK, 0 rows affected (0.07 sec)
  Records: 0  Duplicates: 0  Warnings: 0
  • 修改表名
  # 修改表名
  -- ALTER TABLE tablename RENAME [TO] new_tablename # 语法
  mysql> alter table emp rename employee;
  Query OK, 0 rows affected (0.02 sec)
